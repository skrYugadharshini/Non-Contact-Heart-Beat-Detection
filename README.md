# Non-Contact-Heart-Beat-Detection
This project focuses on non-contact heart rate detection using video data, eliminating the need for physical sensors. The system extracts remote photoplethysmography (rPPG) signals from facial videos and uses a deep learning-based CNN-LSTM model to estimate heart rate accurately.


🚀 Key Features:
Multiple autonomous agents with simple decision logic.

Constraint handling (e.g., task allocation, resource limits, time).

Communication between agents (optional).

Scalable and easy to extend.
